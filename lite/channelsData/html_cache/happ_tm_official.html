Как и обещал, рассказываю про Advanced фрагментацию. Она реализована на известном проекте byedpi (GitHub: hufrea/byedpi) и поддерживает практически все параметры (iOS может некоторые может не поддерживать).

🧩 Как включить Advanced-фрагментацию

Вы можете активировать её:

* глобально — для всего приложения («для всех конфигураций»);
* локально — только для конкретного сервера.

Есть два способа включения:

1. Через интерфейс приложения.
2. Через дополнительные параметры внутри конфигурации сервера.

Параметр называется advancedfragment и может быть использован в таких типах: VLESS, VMess, Trojan.

Пример:


advancedfragment=LS10bHNyZWMgMCtzbSAtZCAwOjI6MytzbQ==


Значение параметра — это строка в формате Base64. Результат расшифровки представляет собой набор флагов, например:


--tlsrec 0+sm -d 0:2:3+sm


Для чего это нужно?

Проект byedpi — довольно сложный, но очень мощный инструмент. Advanced‑фрагментация позволяет гибко управлять фрагментацией трафика и помогает обходить сложные DPI‑системы. Может эффективно решить многие сетевые задачи.

Пример конфигурации


vless://1fb46fdc-e3e4-35d1-bd46-685d775b5762@55.55.55.55:443?encryption=none&node_id=482&headerType=none&type=tcp&security=reality&sni=booking.com&fp=chrome&pbk=YqHW8a4iAc1SZYpTrFVoOQg1F3yAdX1tWXuROZUCsEU&sid=6ba85179e30d4fc2&flow=xtls-rprx-vision&advancedfragment=LS10bHNyZWMgMCtzbSAtZCAwOjI6MytzbQ==&xtls=2#name

Happ для Android TV — это то же приложение, что и для мобильных устройств. Его можно установить как через APK-файл, так и из Google Play.

При первом запуске приложение предложит добавить подписку по локальной сети через QR-код. Просто отсканируйте QR-код в мобильном приложении Happ для iOS или Android, после чего телефон попытается передать выбранные серверы или подписку на телевизор. Если сделать это по локальной сети не удастся, приложение предложит отправить подписку через интернет.

Кроме того, передать подписку или конфигурацию серверов на TV можно через сайт tv.happ.su. На телевизоре выберите пункт Web Input и введите отображённый код либо отсканируйте QR-код и откройте ссылку в браузере телефона.

API ещё в разработке. Как только мы его запустим, я обязательно сообщу.
